### ** Arquitectura Implementada**

#### **Módulos Principales:**
1. ** BASE DE DATOS** - Fuente única de verdad
2. ** ANÁLISIS UNIVARIADO** - Estadísticas individuales
3. ** ANÁLISIS MULTIVARIADO** - Relaciones entre variables
